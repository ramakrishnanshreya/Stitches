# Stitches

Stitches is a concept music-social app for turning favorite lyric moments into shareable visual patches. Users can save short lyric fragments, arrange them into a stitched sequence, and build an emotional narrative through music instead of plain text.

The project explores how music discovery, self-expression, and lightweight social interaction can come together in a minimal black-and-white interface inspired by fabric labels, thread, and hand-stitched collage.

## Project Overview

Stitches is designed around the idea that a single lyric can capture a feeling, but a sequence of lyrics can tell a story. The app lets users collect lyric patches, reorder them, and combine them into a stitch that reflects a mood, memory, relationship, or moment.

The experience is intentionally visual and tactile. Patches appear as cards, stitches behave like horizontal collections, and the interface uses strong contrast, simple navigation, and subtle stitch-inspired details.

## Core User Flow

1. Create an account with full name, email, username, and password.
2. Browse lyric patches from the discovery feed.
3. Select patches and arrange them in a chosen order.
4. Create a stitch from the selected lyric patches.
5. Like, share, comment on, or revisit stitches from the profile and feed screens.

## Key Features

- Account creation and login flow
- Lyric patch browsing
- Horizontal stitch card collections
- Patch ordering for stitch creation
- Like, share, and comment actions
- Profile view for user-created patches and stitches
- Black-and-white visual system with stitch-inspired borders and dividers
- Mobile-first app concept suitable for Flutter implementation

## Design Direction

The visual language is based on:

- High-contrast black and white surfaces
- Monochrome cards with thin borders
- Dashed stitch lines and label-like UI elements
- Compact mobile navigation
- Editorial typography for lyric content
- Minimal controls that keep the focus on the selected lyrics
