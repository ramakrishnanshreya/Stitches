# Stitches

Stitches is a concept music-social app for turning favorite lyric moments into shareable visual patches. Users can save short lyric fragments, arrange them into a stitched sequence, and build an emotional narrative through music instead of plain text.

The project explores how music discovery, self-expression, and lightweight social interaction can come together.

## Project Overview

Stitches is designed around the idea that a single lyric can capture a feeling, but a sequence of lyrics can tell a story. The app lets users collect lyric patches, reorder them, and combine them into a stitch that reflects a mood, memory, relationship, or moment.

The experience is intentionally visual and tactile. Patches appear as cards, stitches behave like horizontal collections, and the interface uses strong contrast, simple navigation, and subtle stitch-inspired details.

## Core User Flow

1. Browse lyric patches from the discovery feed.
2. Select patches and arrange them in a chosen order.
3. Create a stitch from the selected lyric patches.
4. Like, share, comment on, or revisit stitches from the profile and feed screens.
